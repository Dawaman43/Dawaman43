## How to Contribute

### Reporting Issues
If you find any broken links, display issues, or have suggestions for improvements:
1. Open an issue describing the problem or suggestion
2. Include screenshots if applicable
3. Provide specific details about what you'd like to see changed

### Suggesting Enhancements
Feel free to suggest:
- New sections or features for the profile
- Better animations or visual elements
- Improved organization or layout
- Additional badges or stats displays

### Making Changes
1. Fork the repository
2. Create a new branch for your changes (`git checkout -b feature/your-feature-name`)
3. Make your changes
4. Test that the README.md renders correctly
5. Commit your changes with clear, descriptive messages
6. Push to your fork
7. Open a Pull Request with a clear description of your changes

## Style Guidelines

- Keep the existing theme and visual style consistent
- Ensure all external links work correctly
- Test that images and GIFs load properly
- Maintain the existing color scheme (pink #F75C7E and cyan #00FFFF)
- Keep the Naruto/Matrix/Hacker theme intact

## Questions?

Feel free to open an issue if you have any questions about contributing!
